# 简介
从 bugsplat 平台拉取 crash dump 文件，并使用 windbg 工具分析其中的信息。

# 项目搭建
wpf 项目，使用 MvvmLight 框架，使用 nuget 管理 nuget 包。

# Bugsplat api 接口
[接口文档](https://docs.bugsplat.com/introduction/development/web-services/paging-filtering-and-grouping)
